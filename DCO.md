# Developer Certificate of Origin

This project uses a DCO-style contribution convention.

By adding a `Signed-off-by` line to a commit message, the contributor certifies
that they have the right to submit the contribution under the project license.

Use:

```sh
git commit -s
```

which appends a trailer like:

```text
Signed-off-by: Hideaki Igarashi <hide@5stm.net>
```

If AI tools materially assisted the commit, add an `Assisted-by` trailer above the
`Signed-off-by` trailer:

```text
Assisted-by: ChatGPT GPT-5.5 Thinking
Signed-off-by: Hideaki Igarashi <hide@5stm.net>
```

The `Signed-off-by` trailer must be added by a human contributor, not by an AI
tool or automated agent.
